# Wanderlust (Airbnb Clone)

## Overview
This project is a full-stack web application designed to perform basic **CRUD operations** and manage user authentication through **login**, **logout**, and **signup** functionality.

## Features
- **CRUD Operations**:
  - Create: Add new data (e.g., items, posts).
  - Read: View existing data.
  - Update: Edit data.
  - Delete: Remove data.
- **User Authentication**:
  - Secure signup form with password validation.
  - Login system with token-based authentication.
  - Logout functionality to terminate sessions.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (with optional React.js).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB using Mongoose.
- **Authentication**: JWT (JSON Web Tokens).
